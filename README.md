# Проект "shopping-list"

## Описание проекта
Данный проект создан для быстрого создания всевозможных списков

## Используемые технологии
- Python для создания программы проверки паролей.

## Пример запуска

Для запуска скрипта у вас уже должен быть установлен Python3.

Для запуска программы, необходимо записать:

```
python main.py --list
```
Проект выполнен в образовательных целях на онлайн-курсе "Основы Python" школы "Лидер".
