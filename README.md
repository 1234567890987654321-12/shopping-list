# Проект "shopping-list"

## Описание проекта
Простой и эффективный инструмент на Python для создания и управления списками покупок.
## Используемые технологии
- Python3.

## Инструкция запуска
1. скачайте проект
2. через терминал запустите скрипт вписав в него эту команду
   ```
   python main.py
   ```

## Функционал и использование
* add: Добавляет новый товар в ваш список.
* remove: Удаляет существующий товар.
* view: Выводит текущее содержимое списка в консоль
* write: Объединяет все элементы сохраняя из в файл list.txt и завершает работу программы.
